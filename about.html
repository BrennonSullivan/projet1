We are a group of future developers from the Rice University Coding Bootcamp, and this is the first project that we have done since we started! If you click on any of our names, it will take you to our Github page, where we have other repositories that we have worked on. This webpage was created from scratch, and we could not be more excited to show many things that we have learned through the bootcamp. This app lets you search through the entire library of google books, as well as give you suggestions of other books or related movies that might be something you are interested in based on your search. There is also a tab titled "Saved Searches" which saves all your recent searches in case you need to go back to one! On the left side of that tab, there is also a recycle icon, which will refresh the page for you, and delete you search history if it ever gets too cluttered.

apis:
  • Google Books API
  • Movie Database API
  • Taste Dive API
